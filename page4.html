Λογισμικό για τη σχολή

Όλα τα απαραίτητα προγράμματα για τη σχολή είναι συγκεντρωμένα εδω.

Microsoft Windows

Η σχολή συνεργάζεται με τη Microsoft και παρέχει δωρεάν windows education license καθως και δίαφορα άλλα microsoft utilities. Για να πάρετε τα licenses, πρέπει να κάνετε login με το ακαδημαικό σας λογαριασμό στο παρακάτω σύνδεσμο.

Free Microsoft software

Office suite

Σε όσα μαθήματα έχουν εργαστήριο, είναι πολυ πιθανό να πρέπει να γράψετε εργασίες η να κάνετε μια παρουσίαση. Θα χρειαστείτε οπωςδήποτε ε΄να office suite 3 δωρεάν επιλογές είναι το OpenOffice, LibreOffice και OnlyOffice (για όλα τα λειτουργικά). Αν θέλετε κατι πιο αξιόπιστο και πιθανότατα πιο συμβάτο με ορισμένους καθηγητές, υπάρχει και το Microsoft office (μόνο για windows). Δίνεται εικόνα .iso την οποία μπορείτε να την κάνετε mount με windows 10, η daemon tools light.

Download OpenOffice Download LibreOffice Download OnlyOffice Download Microsoft office 2019

C/C++ compilers

Για τα μαθήματα προγραμματισμού "Προγραμματισμός υπολογιστών" και "Αντικειμενοστρεφείς προγραμματισμος" θα χρειαστείτε C++ compiler.

Αν έχετε Windows :
Εύκολη λύση : Χρησιμοποιείτε το Dev-C++

Αν είστε πιο έμπειρος χρήστης μπορείτε να δοκιμασετε το VS και το CLion
*Για το CLion θα χρειαστεί να εγγραφείτε με τον ακαδημαικό σας λογαριασμο, μιας και δεν είναι δωρεαν.

Αν έχετε Linux :
Εύκολη λύση : Χρησιμοποιείτε τα GNU compilers (G++, GCC) με ένα καλό editor οπως το VScode.

Download Dev-C++ Download Visual Studio Download VS code Download CLion

Java IDE (Netbeans)

Για τα μαθήματα "Μεθοδολογίες ανάπτυξης εφαρμογών" και "Δικτυακός προγραμματισμός" θα χρειαστείτε ένα Java IDE, οι καθηγητές προτείνουν το Netbeans 8.2 λόγω ευκολίας χρήσης και συμβατότητας. Επίσης είναι διαθέσιμο για όλα τα λειτουργικά συστήματα.

Download Netbeans 8.2

Multisim

Για τα μαθήματα "Θεωρία κυκλωμάτων", "Ηλεκτρονική", "Ψηφιακή σχεδίαση", "Τεχνολογία υπολογιστικών συστημάτων", "Μικροηλεκτρονική" θα χρειαστείτε τον hardware simulator Multisim ο οποίος τρέχει μόνο σε Windows. 

Download Multisim 14.2 Video tutorial

OpenVPN

Η σχολή παρέχει δωρεάν VPN για φοιτητές, μπορεί να χρησιμοποιηθεί για ανωνυμία αλλα και για την ενεργοποίηση του λογισμικού matlab.

VPN setup instructions

Matlab

Το λογισμικό matlab είναι απαραίτητο για τα μαθήματα "Σηματα και συστήματα", "Ψηφιακή επεξεργασία σήματος" και υπάρχει δωρεάν license απο τη σχολή.
Αν δέν θελετε να κατεβάσετε το matlab στον υπολογιστή σας, υπάρχει και alternative web app, ονόματι Octave.

*Σημείωση : για να ενεργοποιήσετε το Matlab πρέπει να είστε συνδεδεμένοι στο VPN της σχολής.

Get Matlab from Uniwa Go to Octave

Cisco Packet tracer

Για το μάθημα "Δίκτυα υπολογιστών ΙΙ" θα χρειαστείτε το λογισμικό Packet tracer.
*Παρακάτω δίνεται η έκδοση 7.3.0, η οποία δεν απαιτεί login οπως οι τελευταίες.

Download Cisco Packet tracer

Python & IDEs

Στα μαθήματα "Τεχνητη νοημοσύνη", "Επεξεργασία εικόνας", "Ανάκτηση δεδομένων" θα χρησιμοποιήσετε την γλώσσα python3, προτεινόμενα IDEs είναι το spyder και το pycharm. Αν ο υπολογιστής σας δεν έχει εγκατεστημένη την python3, πρέπει να την κατεβάσετε και αυτην.

Download Python 3 Download Spyder Download Pycharm

CLIPS

Για το δεύτερο εργαστηριακό μέρος του μαθήματος "Τεχνητη νοημοσύνη" θα χρησιμοποίησετε το λογισμικο CLIPS για την δημιουργία expert systems. (Διαθέσιμο μόνο για Windows)

Download CLIPS

MARS (MIPS emu)

Στο μάθημα "Αρχιτεκτονική Υπολογιστών" θα χρειαστείτε τον MIPS emulator 'Mars' ο οποίος είναι γραμμένος σε Java (χρειάζεστε JRE/JDK για να τρέξει).

Download MARS

Modelsim

Στο μάθημα "Σχεδιασμός ψηφιακών συστημάτων" θα χρειαστείτε το εργαλείο Modelsim. Στη σχολή προτείνετε η δωρεάν έκδοση quartus prime lite. Είναι διαθέσιμο για Linux και Windows.

Download Modelsim

Virtual Machines

Είναι πολυ πιθανό να χρειαστείτε VMs για πολλά μαθήματα της σχολής, αν επιλέξετε να μην κάνετε dual boot. Για windows, κύριες λύσεις αποτελούν το VMware και το Virtual box που είναι εύκολα στη χρήση. Τα συγκεκριμένα είναι L2 hypervisors (τρέχουν στο υπάρχον OS), υπαρχουν και τα L1, αλλα αυτα δεν προτείνονται γιατι ειναι περίπλοκα στη χρήση.

Download VMware Player Download VirtualBox
